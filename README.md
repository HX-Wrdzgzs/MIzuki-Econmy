# Mizuki Economy (25时经济系统)

mizuki_econmy 是一款为 NoneBot2 框架设计的群聊经济与社交互动插件。本项目针对 QQ 官方机器人的底层 API 进行了深度定制，在保证高并发数据安全的前提下，提供了丰富的经济体系与极具表现力的原生 UI 交互。

## 核心特性

- 原生官方 UI 深度适配
  - 动态降级策略：通过特判 bot.self_id，对官方机器人实例下发原生的 Markdown 与 内联按钮；对非官方实例自动执行平滑降级，以纯文本形式输出，保障多实例兼容性。
  - 规范化组件：基于 lib_msg.py 抽象规范，统一采用一维扁平化列表构建按钮，自动处理布局排版与跳转逻辑。
- 异步图像渲染
  - 基于 Pillow 库实现全量 UI 图像化，涵盖签到面板、个人信息卡片、打工作业结算及背包概览。
  - 采用瑞希定制色系 UI 规范，自动拉取并绘制用户 QQ 头像（支持 mapid 映射与 API 获取）。
  - 支持个人自定义背景图片，发送设置背景附带图片即可自适应拉伸并叠加半透明主题蒙版。
  - PJSK卡牌原画自适应 16:9 比例无拉伸裁切展示，十连抽卡生成精美紧凑的网格视图，卡片高度由 560 压缩至 430 以节约带宽。
- 事务级并发控制
  - 底层基于 aiomysql，全面启用 InnoDB 引擎。
  - 在核心资金流转（如抢红包、集资扣款）中严格遵循 FOR UPDATE 行级锁与事务隔离机制，彻底杜绝高并发场景下的超卖与死锁异常。
- 高并发社交应用
  - 拼手气红包：引入微信同款二倍均值算法进行资金拆分，采用内存队列承载抢夺请求，内置系统税收流转。
  - V50 全服大乐透：结合 nonebot-plugin-apscheduler 实现跨群定时任务。周三/周四自动触发预热广播，周四正午自动截断报名池并进行全网一致性开奖。
- 完善的经济模型
  - 内置运势签到、多维度体力打工、等级跃升系统、卡牌三合一合成系统以及支持限制购买条件的道具商城。

## 依赖

- nonebot2 >= 2.0.0
- nonebot-adapter-onebot
- nonebot-plugin-apscheduler
- aiomysql
- Pillow
- httpx

## 环境配置

在 NoneBot2 项目的 .env 或 .env.prod 文件中配置以下数据库配置项：

```env
SIGN_MYSQL_HOST=127.0.0.1
SIGN_MYSQL_PORT=3306
SIGN_MYSQL_USER=your_username
SIGN_MYSQL_PASSWORD=your_password
SIGN_MYSQL_DB=your_database_name
```

如果未连接数据库，系统会自动切换到本地降级存储模式，数据将被保存在项目根目录的 data 文件夹下：
`data/mizuki_econmy/local_db.json`

## 指令列表

### 1. 基础与娱乐
- 签到: 每日签到并获取奖励
- 查询 / 个人信息 / 余额: 查看个人余额与详细信息
- 打工: 消耗体力进行委托任务
- 排行榜: 查看群内财富排行榜
- 背包 / 我的背包: 查看自己拥有的道具
- 使用 [物品ID]: 消耗并使用背包中的物品
- 每日问答: 参与日常趣味答题
- 设置背景 [发送图片]: 自定义个人中心背景
- 兑换码 [CDK]: 兑换特殊系统礼包
- 骰子 [下注]: 摇双骰娱乐小游戏
- 幸运数字 [数字] [下注]: 猜幸运数字小游戏

### 2. 交互与社交
- pk @某人: 发起代币对决PK
- 接受pk / 拒绝pk: 应对他人的PK决斗请求
- 猜拳 @某人 金额: 石头剪刀布决斗
- 发红包 [金额] [份数]: 在群里塞红包
- 抢红包: 抢夺群内发出的最新红包
- v50报名: 参与周四的v50大奖集资
- 转账 @某人 [金额]: 转账给指定用户
- 写日记 [内容]: 在日记墙留下实名心情
- 树洞 [内容]: 在日记墙留下匿名心情
- 共鸣 [日记ID]: 产生共鸣并为作者提供奖励
- 日记墙: 查看群友写的日记列表

### 3. 经济与经营
- 购买 [物品ID]: 网页或命令快捷购买商城物资
- 贷款 [金额]: 向系统借贷代币
- 还款: 还清当前的系统借贷
- 股市: 查看并交易虚拟股票
- 卖出 [股票ID] [数量]: 变现持有的股票
- 我的持仓: 查看个人股票资产
- 拍卖 列表: 查看拍卖行上架物品
- 每日任务: 查看今天可完成的任务
- 领取奖励 [任务ID]: 结算完成的任务奖励

### 4. 收集与育成
- 单抽: 消耗代币抽取一张卡牌
- 十连: 一次性抽取十张卡牌
- 我的卡牌: 查看个人卡牌馆藏
- 合成 [星级]: 三合一升级高星卡牌
- 农场: 查看自己的作物种植情况
- 种植 [作物ID]: 播种相应植物种子
- 收菜: 收获已经成熟的农作物
- 宠物: 查看个人宠物状态
- 领养宠物 [类型]: 获得自己的新宠物
- 喂食: 消耗代币给宠物喂食
- 副本: 消耗代币组队挑战Boss
- 探索 [区域ID]: 探索不同地图获取产出
- 成就墙: 展示已解锁的系统成就
